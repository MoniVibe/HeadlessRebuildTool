# Evidence Order

Use this strict order when diagnosing:

1. `meta.json`
2. `out/run_summary.json`
3. `out/watchdog.json`
4. `out/player.log`
5. `out/stdout.log` and `out/stderr.log`

Prefer root cause evidence over symptom tails. If the same signature appears twice, consult:

- `Polish/Docs/ANVILOOP_RECURRING_ERRORS.md`
